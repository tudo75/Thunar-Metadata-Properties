pixbuf_from_sample

Description:

public Pixbuf? pixbuf_from_sample (Sample sample)

Converts a GStreamer Sample (used by Audio/Video tags) into a Gdk.Texture. Extracts the buffer, creates a MemoryInputStream, loads a Pixbuf, then a Texture.

Parameters:

sample

The Gst.Sample containing the image data.

Returns:

A Gdk.Pixbuf if successful, null otherwise.
