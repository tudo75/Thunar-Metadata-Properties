create_list_row

Description:

public static void create_list_row (Grid grid, ref int row, string label_text, List<string> items, bool use_markup = false)

Creates a list row in the provided grid.

Parameters:

grid

The Gtk Grid to which the row will be added.

row

A reference to the current row index, which will be incremented.

label_text

The text for the label (bold).

items

A list of strings to be displayed as items in the row.

use_markup

A boolean indicating whether to use markup in the item labels.
