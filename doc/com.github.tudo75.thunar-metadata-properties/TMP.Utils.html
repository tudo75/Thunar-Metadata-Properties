Classes:

  • UiHelper - UiHelper: A utility class for creating and managing UI components in a Gtk application. This class provides static methods to create labeled rows, list rows, section headers, and separators within a Gtk Grid layout. Methods: - create_label_row: Creates a row with a bold label and a value. - create_list_row: Creates a row with a bold label and a list of items, supporting markup. - add_section_header: Adds a section header to the grid. - add_separator: Adds a horizontal separator to the grid.
